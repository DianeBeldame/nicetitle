# nicetitle
Rstudio Addin - create boxed title in an Rscript

## Installation

```R
# install.packages("devtools")
devtools::install_github("ThinkRstat/nicetitle")
```
## usage

```R
# write a sentense, select it an go in Addins > Nice Title

this is a sentense

will be transformed as :

############################################################
#                                                          #
#                    this is a sentense                    #
#                                                          #
############################################################

```
